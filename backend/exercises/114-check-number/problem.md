# Check Number

Today, Ty learned about four-digit natural numbers. Before ending the lesson, the teacher gave Ty a homework assignment:

“Given a positive four-digit integer N, check whether the sum of the first two digits of N is equal to the sum of the last two digits.”

Write a program to help Ty solve this task.

## Input
A single positive four-digit integer `N`

## Output
Print `"YES"` if the sum of the first two digits equals the sum of the last two digits, otherwise print `"NO"`

## Sample Input
1937

## Sample Output
YES

## Sample Input
9991

## Sample Output
NO
