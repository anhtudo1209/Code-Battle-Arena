# Longest Substring Without Repeating Characters

Write a function int lengthOfLongestSubstring(char *s) that finds the length of the longest substring within a given string (char array) that contains no repeating characters.

## Input
A string

## Output
Finds the length of the longest substring within a given string (char array) that contains no repeating characters.

## Sample Input
abcabcbb

## Sample Output
3
