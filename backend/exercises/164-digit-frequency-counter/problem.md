# Digit Frequency Counter

Given a string s, count how many times each digit character '0' through '9' appears in the string.

## Input
A single string s.

## Output
A single line containing 10 integers.  
The i-th integer represents how many times digit i - 1 appears in the string.

## Sample Input
ab8 jf0kl0 fsf1 fsff9 lort 5 klwr9 klkf6

## Sample Output
2 1 0 0 0 1 1 0 1 2
