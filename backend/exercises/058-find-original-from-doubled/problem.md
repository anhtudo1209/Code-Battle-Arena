# Find Original From Doubled Array

An array original was transformed by doubling each element to create doubled. This array was then randomly shuffled. Given doubled, find and return original. If doubled is not a valid doubled array, return an empty array.

## Input
An array

## Output
Given doubled, find and return original. If doubled is not a valid doubled array, return an empty array.

## Sample Input
1 3 4 2 6 8

## Sample Output
1 3 4
