# Spiral Matrix

Given an integer n, print an n x n matrix filled with numbers from 1 to n*n in spiral order

## Input
An integer n

## Output
An n x n matrix filled with numbers from 1 to n*n in spiral order.

## Sample Input
3

## Sample Output
1 2 3	
8 9 4	
7 6 5

## Sample Input
4

## Sample Output
1 2 3 4
12 13 14 5
11 16 15 6
10 9 8 7
