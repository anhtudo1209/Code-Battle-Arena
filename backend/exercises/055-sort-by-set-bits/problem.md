# Sort by Set Bits

Given an array of integers, sort them in ascending order based on the number of '1' bits in their binary representation. If two numbers have the same bit count, sort them by their original value.

## Input
An array of integers

## Output
Sort them in ascending order based on the number of '1' bits in their binary representation

## Sample Input
7
0 1 2 3 4 5 6

## Sample Output
0 1 2 4 3 5 6
