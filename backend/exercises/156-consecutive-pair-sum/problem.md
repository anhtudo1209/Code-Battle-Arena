# Consecutive Pair Sum

Given a sequence of N integers A1, A2, ..., AN (1 <= N <= 10^3) and an integer X.  
Count the number of consecutive pairs in the sequence whose sum equals X.

## Input
- Line 1: two integers N and X  
- Line 2: N integers A1, A2, ..., AN

## Output
- A single integer: the number of consecutive pairs whose sum equals X

## Sample Input
5 7
6 1 9 -2 9

## Sample Output
3
