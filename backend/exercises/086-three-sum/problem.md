# Three Sum

Given an array of integers, find all unique triplets (a, b, c) in the array that sum to zero.

## Input
Line 1: the size of the array
Line 2: the array

## Output
Find all unique triplets (a, b, c) in the array that sum to zero.

## Sample Input
6
-1 0 1 2 -1 -4

## Sample Output
-1, -1, 2 
-1, 0, 1
