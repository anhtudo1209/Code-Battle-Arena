# Prime factorization
In this problem, we would like to perform a prime factorization of a given natural number N: N = 120 = 2 * 2 * 2 * 3 * 5; 
N = 84 = 2 * 2 * 3 * 7 
Write a code to implement the factorization

## Input
A natural number N

## Output
Perform a prime factorization of a given natural number N

## Sample Input
84


## Sample Output
2 2 3 7
