# Sum of Two Numbers

Given a sorted array of integers and a target value, return the index of the target. If the target is not found, return -1. You must implement this using a while loop.

## Input
A sorted array of integers and a target value

## Output
Return the index of the target. If the target is not found, return -1.

## Sample Input
5
10 20 30 40 50
30

## Sample Output
2
