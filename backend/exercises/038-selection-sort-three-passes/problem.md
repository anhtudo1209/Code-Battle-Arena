# Selection Sort - Three Passes

Given an array of integers, perform 3 passes of Selection Sort and print the array after each pass.

## Input
An array of integers

## Output
The array after each of the 3 passes of Selection Sort.

## Sample Input
5
5 3 4 1 2

## Sample Output
2 3 4 1 5
2 3 1 4 5
2 1 3 4 5
