# Multiplication Table

Input a number n, then print its multiplication table up to 10.


## Input
A number n

## Output
its multiplication table up to 10.

## Sample Input
7

## Sample Output
7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
7 x 4 = 28
7 x 5 = 35
7 x 6 = 42
7 x 7 = 49
7 x 8 = 56
7 x 9 = 63
7 x 10 = 70
