# Triplets

Given a sequence of n numbers A1, A2, ..., An.  
Count the number of triplets in these n numbers such that the three numbers can form the sides of a triangle, and also find the largest perimeter among all such triangles.

## Input
- Line 1: integer N (1 <= n <= 100)  
- Line 2: N integers A1, A2, ..., AN

## Output
- The number of triplets and the largest perimeter satisfying the condition

## Sample Input
4
1 2 4 3

## Sample Output
1 9
