# Prime-Sum Pair Counter

Given a sequence of N integers A1, A2, ..., AN (1 <= N <= 1000).  
Count the number of pairs (not necessarily adjacent) whose sum is a prime number.

## Input
- Line 1: Integer N
- Line 2: N integers A1, A2, ..., AN

## Output
A single integer: the number of pairs whose sum is a prime number.

## Sample Input
5
9 6 -2 9 1

## Sample Output
3
