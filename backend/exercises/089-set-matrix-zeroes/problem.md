# Set Matrix Zeroes

Write a function void setZeroes(int matrix[M][N]) that takes an M x N matrix. If an element is 0, set its entire row and column to 0. This must be done in-place with O(1) extra space.

## Input
An M x N matrix.

## Output
If an element is 0, set its entire row and column to 0.

## Sample Input
3 3
1 1 1
1 0 1
1 1 1


## Sample Output
1 0 1
0 0 0
1 0 1
