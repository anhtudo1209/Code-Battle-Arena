# Character Position Finder

Given a character x and a string s, count how many times x appears in s and list all positions where it occurs.

## Input
- Line 1: character x  
- Line 2: string s

## Output
- Line 1: number of occurrences of x in s  
- Line 2: positions (0-based) where x appears

## Sample Input
e  
ecek e de

## Sample Output
4  
0 2 5 8
