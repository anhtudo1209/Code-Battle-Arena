# String Replacement

Given three strings s1, s2, and s, replace every occurrence of s1 in s with s2.

## Input
- Line 1: string s1
- Line 2: string s2
- Line 3: string s

## Output
- The string s after replacing all occurrences of s1 with s2.

## Sample Input
anh
em
anh chao em va em chao lai anh

## Sample Output
em chao em va em chao lai em
