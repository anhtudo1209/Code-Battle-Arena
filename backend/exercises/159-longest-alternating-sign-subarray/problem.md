# Longest Alternating Sign Subarray

Given a sequence of N integers A1, A2, ..., AN (1 ≤ N ≤ 10⁶),  
find the longest contiguous subarray whose elements alternate in sign.

## Input
- Line 1: the integer N  
- Line 2: N integers A1, A2, ..., AN

## Output
- A single integer: the length of the longest alternating-sign contiguous subarray

## Sample Input
5  
6 67 -8 48 88

## Sample Output
3
