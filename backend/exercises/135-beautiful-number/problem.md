# Beautiful Number

In Eastern culture, a "beautiful number" is a natural number that contains only the digits 6 and 8.  
For example: 6, 8, 66, 68, 86, 88, 886, 668, … are beautiful numbers;  
468, 5, 728 … are not beautiful numbers.

Given a positive integer N, check whether N is a beautiful number.

## Input
A positive integer N

## Output
Print "YES" if N is a beautiful number, otherwise print "NO"

## Sample Input
8686

## Sample Output
YES

## Sample Input
876

## Sample Output
NO
