# Group Anagrams

Given an array of strings, group the anagrams together. Return a list of lists.

## Input
An array of strings, group the anagrams together

## Output
Return a list of lists

## Sample Input
["eat", "tea", "tan", "ate", "nat", "bat"]

## Sample Output
[["bat"], ["nat", "tan"], ["ate", "eat", "tea"]]
