# Insertion Sort - Three Passes

Given an array of integers, perform 3 passes of Insertion Sort and print the array after each pass.

## Input
An array of integers

## Output
The array after each of the 3 passes of Insertion Sort.

## Sample Input
[7, 3, 5, 2, 9]

## Sample Output
[3, 7, 5, 2, 9]
[3, 5, 7, 2, 9]
[2, 3, 5, 7, 9]
[2, 3, 5, 7, 9]
