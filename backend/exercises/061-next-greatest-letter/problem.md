# Next Greatest Letter

Given a sorted array of characters letters and a character target, find the smallest character in letters that is lexicographically greater than target. If no such character exists, return the first character.

## Input
An array of characters letters and a character target

## Output
Find the smallest character in letters that is lexicographically greater than target. If no such character exists, return the first character.

## Sample Input
c 
a f j

## Sample Output
f
