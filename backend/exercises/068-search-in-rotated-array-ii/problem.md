# Search in Rotated Array II

This is the same as "Search in Rotated Array," but the array may contain duplicates. Return true if the target is found, false otherwise.

## Input
An array

## Output
Return true if the target is found, false otherwise

## Sample Input
target = 0, nums = [2, 5, 6, 0, 0, 1, 2]

## Sample Output
true
