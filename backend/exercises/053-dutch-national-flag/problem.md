# Dutch National Flag

Given an array containing only 0s, 1s, and 2s, write a function to sort it in O(n) time without using any extra space (Dutch National Flag problem).

## Input
An array

## Output
Sort array in O(n) time without using any extra space

## Sample Input
2 0 2 1 1 0


## Sample Output
0 0 1 1 2 2
