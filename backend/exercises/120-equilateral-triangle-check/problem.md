# Equilateral Triangle Check

Enter three numbers a, b, c. Check whether the three numbers can form the sides of an equilateral triangle.

## Input
A single line containing three numbers a, b, c

## Output
Print "YES" if a, b, c can form an equilateral triangle, otherwise print "NO"

## Sample Input
3 4 5

## Sample Output
NO

## Sample Input
5 5 5

## Sample Output
YES
