# Common Characters

Given two strings X and Y consisting only of lowercase Latin letters ('a' to 'z'), output the set of characters that appear in both X and Y, sorted in alphabetical order.  
Each common character should be printed exactly once.  
It is guaranteed that there is at least one common character.

## Input
- Line 1: string X  
- Line 2: string Y

## Output
- The list of common characters between X and Y, one character per line, sorted from 'a' to 'z'.

## Sample Input
abbc
bbbc

## Sample Output
b
c

## Sample Input
abbc
bbbcaa

## Sample Output
a
b
c

## Sample Input
aebbc
ebbbc

## Sample Output
a
b
e
