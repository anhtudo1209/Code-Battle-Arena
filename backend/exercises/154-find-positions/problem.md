# Find Positions

Given a sequence of n real numbers. Then:  
- Find the first negative number in the sequence  
- Find the last positive number in the sequence  
- Find the first and last positions of the element x in the sequence

## Input
- Line 1: two positive numbers n and x (1 <= n <= 100)  
- Line 2: n real numbers a1, a2, ..., an

## Output
- Line 1: the value of the first negative number, the value of the last positive number  
- Line 2: the first and last positions of the element x in the sequence

## Sample Input
7 2.00
-1.00 4.00 -3.00 2.00 -5.00 2.00 -1.00

## Sample Output
-1.00 2.00
4 6
