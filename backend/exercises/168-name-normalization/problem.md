# Name Normalization

Given a full name string s, normalize it so that:

- There are no extra spaces (only one space between words, no leading/trailing spaces).
- The first letter of each word is uppercase, and the remaining letters are lowercase.

## Input
A single string s representing a person's name.

## Output
The normalized full name.

## Sample Input
       nGuyEN         thANH          tUNG

## Sample Output
Nguyen Thanh Tung
