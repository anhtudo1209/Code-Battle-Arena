# Pair Sum Counter 

Given a sequence of N integers A1, A2, ..., AN (1 ≤ N ≤ 1000) and an integer X,  
count the number of distinct index pairs (i, j) with i < j such that Ai + Aj = X.

## Input
- Line 1: two integers N and X  
- Line 2: N integers A1, A2, ..., AN

## Output
- A single integer: the number of pairs whose sum equals X

## Sample Input
5 7  
9 6 -2 9 1

## Sample Output
3
