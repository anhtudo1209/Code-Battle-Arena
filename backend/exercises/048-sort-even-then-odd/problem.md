# Sort Even Then Odd

Sort the array so that even numbers come first (ascending), followed by odd numbers (ascending).

## Input
An array of integers

## Output
The array with even numbers first (ascending), followed by odd numbers (ascending).

## Sample Input
[7, 2, 4, 9, 1, 6]

## Sample Output
[2, 4, 6, 1, 7, 9]
