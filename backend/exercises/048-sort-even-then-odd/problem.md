# Sort Even Then Odd

Sort the array so that even numbers come first (ascending), followed by odd numbers (ascending).

## Input
The input consists of an integer n, representing the number of elements in the array, followed by n integers representing the elements of the array.

## Output
The array with even numbers first (ascending), followed by odd numbers (ascending).

## Sample Input
6
7 2 4 9 1 6

## Sample Output
2 4 6 1 7 9
