# Single Number

Write a function int singleNumber(int *arr, int size) that takes an array where every element appears twice except for one. Find that single, non-duplicated element. Your solution must be O(n) time and O(1) space.

## Input
An array

## Output
Find that single, non-duplicated element.

## Sample Input
5
5 2 2 1 1

## Sample Output
5
