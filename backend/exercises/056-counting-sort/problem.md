# Counting Sort

Write a function void countingSort(int *arr, int size) that sorts an array of non-negative integers using the Counting Sort algorithm. You may assume the maximum value is known (e.g., 100)

## Input
An array

## Output
Sorts an array of non-negative integers using the Counting Sort algorithm.

## Sample Input
4 2 2 3 3 1

## Sample Output
1 2 2 3 3 4
