# Range Sum Query

Implement a class NumArray that has a constructor NumArray(int *arr) which pre-processes an array. It also has a function int sumRange(int left, int right) that returns the sum of the elements from left to right (inclusive) in O(1) time.

## Input
An array

## Output
Returns the sum of the elements from left to right 

## Sample Input
5
1 2 3 4 5
3
0 2
2 4
0 4


## Sample Output
6
12
15
