# Sum of Four Numbers

Enter four integers `a`, `b`, `c`, `d` from the keyboard. Then display the sum of these four numbers and also output the tens and units digits of the sum.

## Input
A single line containing four integers `a`, `b`, `c`, `d`

## Output
- Line 1: The sum of the four numbers  
- Line 2: The tens and units digits of the sum, separated by a space

## Sample Input
99 1 2 3

## Sample Output
105
0 5
