# Rectangular Brick

A brick has the shape of a rectangular box with dimensions a, b, c. Determine whether the brick can pass through a rectangular hole with dimensions x and y.

## Input
Five positive integers a, b, c, x, y

## Output
Print "CO" if the brick can pass through the hole; otherwise, print "KHONG"

## Sample Input
3 4 7 4 5

## Sample Output
CO

## Sample Input
5 4 3 3 5

## Sample Output
KHONG
