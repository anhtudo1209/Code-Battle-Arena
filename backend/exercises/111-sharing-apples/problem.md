# Sharing Apples

Grandmother buys `n` apples for Viet and Nam, and she wants to divide them equally between the two brothers.  
If `n` is even, both brothers receive the same number of apples.  
If `n` is odd, Nam will receive 1 more apple than Viet.

Determine the number of apples Viet and Nam will receive.

## Input
A single integer `n`

## Output
Two integers: the number of apples Viet receives and the number of apples Nam receives

## Sample Input
9

## Sample Output
4 5

## Sample Input
8

## Sample Output
4 4
