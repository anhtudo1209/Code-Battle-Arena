# Quick Sort

Write a function void quickSort(int *arr, int low, int high) that sorts an array using the Quick Sort algorithm. You will also need to implement the partition helper function.

## Input
An array

## Output
Sorts an array using the Quick Sort algorithm.

## Sample Input
7
7 2 1 6 8 5 3

## Sample Output
1 2 3 5 6 7 8
