# Sort Complex Numbers by Magnitude

Sort complex numbers by magnitude (distance from origin).


## Input
The input consists of an integer n, representing the number of complex numbers followed by n pairs of integers where each pair represents the real and imaginary parts of a complex number.

## Output
Complex numbers sorted by magnitude.

## Sample Input
3
3 4 
1 1 
0 2

## Sample Output
1 1
0 2
3 4
