# Prime Function

Define a function: f(x) = the sum of all prime numbers less than or equal to x.  
Example: f(4) = 2 + 3 = 5; f(7) = 2 + 3 + 5 + 7 = 17.

Given an integer n from input, check whether f(n) is a prime number.

## Input
An integer n (1 ≤ n ≤ 100000)

## Output
Print "YES" if f(n) is a prime number, otherwise print "NO"

## Sample Input
5

## Sample Output
NO

## Sample Input
7

## Sample Output
YES
