# Rotate Image

You are given an N x N 2D matrix representing an image. Write a function void rotate(int matrix[N][N]) that rotates the image by 90 degrees clockwise in-place (without using another 2D matrix).

## Input
An N x N 2D matrix

## Output
Rotates the image by 90 degrees clockwise in-place

## Sample Input
3 3
1 2 3
4 5 6
7 8 9


## Sample Output
7 4 1
8 5 2
9 6 3
