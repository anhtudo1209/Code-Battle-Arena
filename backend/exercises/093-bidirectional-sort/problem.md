# Rotate Image

Implement a bidirectional (Cocktail Shaker) sort. In each pass, you first do a forward pass (like insertion/bubble sort) from 1 to n, and then a backward pass from n to 1, repeating until the array is sorted.

## Input
An array

## Output
Sorted an array

## Sample Input
5
5 4 3 2 1

## Sample Output
1 2 3 4 5
