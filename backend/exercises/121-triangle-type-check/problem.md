# Triangle Type Check

Enter three numbers a, b, c. Determine whether the three numbers can form a triangle, and if so, identify its type: vuong (right-angled), can (isosceles), deu (equilateral), thuong (scalene).

## Input
A single line containing three numbers a, b, c

## Output
Print one of the following:
- "vuong" if the triangle is right-angled
- "can" if the triangle is isosceles
- "deu" if the triangle is equilateral
- "thuong" if the triangle is a regular (scalene) triangle
- "khong" if a, b, c cannot form a triangle

## Sample Input
3 4 5

## Sample Output
vuong

## Sample Input
5 5 5

## Sample Output
deu
