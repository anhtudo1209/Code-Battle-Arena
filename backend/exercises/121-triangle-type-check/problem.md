# Triangle Type Check

Enter three numbers a, b, c. Determine whether the three numbers can form a triangle, and if so, identify its type: right-angled, isosceles, equilateral, scalene.

## Input
A single line containing three numbers a, b, c

## Output
Print one of the following:
- "right-angled" if the triangle is right-angled
- "isosceles" if the triangle is isosceles
- "equilateral" if the triangle is equilateral
- "scalene" if the triangle is a regular (scalene) triangle
- "no" if a, b, c cannot form a triangle

## Sample Input
3 4 5

## Sample Output
right-angled

## Sample Input
5 5 5

## Sample Output
equilateral
