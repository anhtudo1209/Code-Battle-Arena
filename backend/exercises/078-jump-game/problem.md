# Jump Game

You are given an array of non-negative integers nums. You are initially at index 0. Each element nums[i] is your maximum jump length from that position. Return true if you can reach the last index, false otherwise.

## Input
An array of non-negative integers nums

## Output
Return true if you can reach the last index, false otherwise.

## Sample Input
2 3 1 1 4

## Sample Output
true
