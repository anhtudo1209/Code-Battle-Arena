# Sphenic number check

Write a function bool isSphenic(int n) to verify whether a natural number n is sphenic. A sphenic number is a product of three distinct prime numbers (p x q x r, where p not equal to q not equal to r, and all are prime).

## Input
A natural number

## Output
Verify whether a natural number n is sphenic.

## Sample Input
30


## Sample Output
1
