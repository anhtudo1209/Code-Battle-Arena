# Contains Duplicate

Write a function bool containsDuplicate(int *arr, int size) that returns true if any value appears at least twice in the array, and false if every element is distinct.

## Input
An array

## Output
Returns true if any value appears at least twice in the array, and false if every element is distinct.

## Sample Input
7
7 1 2 3 4 5 1

## Sample Output
true
