# Count Beautiful Numbers

Count the number of beautiful numbers less than or equal to n (n ≤ 100000).  
In Eastern culture, a "beautiful number" is a natural number that contains only the digits 6 and 8.  
For example: 6, 8, 66, 68, 86, 88, 886, 668, … are beautiful numbers;  
468, 5, 728 … are not beautiful numbers.

## Input
A positive integer n

## Output
The number of beautiful numbers less than or equal to n

## Sample Input
80

## Sample Output
4
