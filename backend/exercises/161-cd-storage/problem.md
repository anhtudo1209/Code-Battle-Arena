# CD Storage

You are given N mp3 music files with sizes A1, A2, ..., AN.  
You want to choose as many files as possible to fit onto a CD with capacity S.

## Input
- Line 1: two integers N (1 ≤ N ≤ 100) and S  
- Line 2: N integers A1, A2, ..., AN

## Output
- A single integer: the maximum number of files that can fit on the CD

## Sample Input
5 16  
9 4 4 12 7

## Sample Output
3
