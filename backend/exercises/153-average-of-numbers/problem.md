# Average of Numbers

Given a sequence of n real numbers. Then perform the following tasks:  
- Compute the average of the elements in the sequence  
- Find the element in the sequence whose difference from the average is the smallest

## Input
- Line 1: a positive integer n (1 <= n <= 100)  
- Line 2: n real numbers a1, a2, ..., an

## Output
- Line 1: the average value of the sequence  
- Line 2: the element whose difference from the average is the smallest

## Sample Input
5
-1.00 4.00 -3.00 2.00 5.00

## Sample Output
1.40
2.00
